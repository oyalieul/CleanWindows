CleanWindows
CleanWindows is a Python utility that safely removes unnecessary temporary files, logs, and system cache to free up disk space on Windows machines.

Key Features
Safely cleans junk files from Windows without harming important system files.
Automatically requests administrator privileges via Windows application manifest.
Standalone executable (.exe) — no Python installation required.
Python script included for showcase and reference.

Cleaned Locations
The program removes unnecessary files from the following locations:
C:/Users/{user}/AppData/Local/Temp
C:/Windows/Temp
C:/Windows/Prefetch
C:/Windows/Logs

Usage
Download the CleanWindows.exe from this repository.
Double-click the file to run it. That’s it! The program will automatically clean the junk files.
Other files in this repository are provided for Python script showcase and reference.
